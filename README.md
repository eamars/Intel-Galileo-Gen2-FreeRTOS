# Intel Galileo Gen 2 Bare metal

Description
-----------
The Intel Galileo Gen  2 Bare metal programming example is taken from FreeRTOS official project. This project is built upon CMake with minimal dependencies. 

Requirements
------------
Few dependencies are required to install before building the project

- i686-elf-gcc (tested with GCC-7.1)
- CMake (version 3.8.1 or later)

Build
-----

1. Makefile is required to be generated by CMake.


        chmod +x build_debug.sh
        ./build.sh
        
2. Compile


        make
